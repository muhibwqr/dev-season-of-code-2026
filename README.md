# Dev Season of Code 2026

> Open source contribution tracker and mentorship matching platform for Dev Season of Code 2026.

## What It Does
- **Contribution Tracker** — log PRs, issues, docs, and reviews with automatic point scoring
- **Leaderboard** — real-time rankings across all contributors
- **Mentor Matching** — find mentors by skill (Python, React, ML, etc.)

## Quick Start
```bash
git clone https://github.com/muhibwqr/dev-season-of-code-2026.git
cd dev-season-of-code-2026
pip install -r requirements.txt
python app.py
# Open http://localhost:5000
```

## Point System
| Contribution Type | Points |
|---|---|
| Pull Request | 10 |
| Documentation | 7 |
| Code Review | 6 |
| Issue / Bug Report | 5 |

## Tech Stack
Python, Flask, SQLite, HTML5, CSS3, JavaScript (vanilla)
